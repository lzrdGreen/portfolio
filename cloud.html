My Cloud Journey

Serverless Mini-Projects:

  • Positive Affirmation Generator: This fully serverless application leverages Amazon CloudFront for optimised content delivery, DynamoDB for storing affirmations, Lambda function for processing logic, and API Gateway to expose the functionality. Though modest in scope, this project offered valuable insight into designing and deploying a scalable, serverless stack.

Current Cloud Migration Initiative:

Building on this initial success, I'm taking on a more substantial challenge: migrating my Project Management System to AWS using a lift-tinker-and-shift strategy. This migration focuses on:

  • Implementing containerisation best practices.

  • Utilising AWS's comprehensive DevOps tool chain.

  • Optimising the application architecture for cloud deployment.

This initiative represents my progression from basic cloud concepts to real-world application deployment, while making full use of AWS's free tier resources for development and testing.
